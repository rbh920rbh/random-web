# 随机抽取工具

一个功能完整的本地随机抽取工具，支持手机和PC浏览器使用。

## 功能特性

- ✅ **多列表管理**：创建、选择和删除不同的随机列表
- ✅ **智能记忆**：自动记住上次选择的列表，下次打开时默认选择
- ✅ **权重系统**：为每个项设置权重，权重越高被选中的概率越大
- ✅ **权重递减**：可选功能，上次选中项权重减为1/3，再上次减为2/3
- ✅ **历史记录**：记录最近3次抽取结果
- ✅ **弹窗结果**：美观的弹窗展示抽取结果，支持点击关闭或ESC键关闭
- ✅ **数据导入导出**：支持导出数据为JSON文件，方便备份和共享
- ✅ **本地存储**：数据保存在浏览器本地，同时支持文件导入
- ✅ **响应式设计**：完美适配手机和PC浏览器

## 使用方法

1. **创建列表**：点击"新建列表"按钮，输入列表名称
2. **抽取**：点击"开始抽取"按钮，弹窗显示抽取结果
3. **配置设置**：根据需要启用权重递减功能
4. **查看历史**：查看最近3次抽取结果
5. **管理项**：在页面底部添加、编辑或删除列表项及其权重
6. **数据管理**：
   - **导出数据**：点击"导出数据"按钮，将所有数据保存为JSON文件
   - **导入数据**：点击"导入数据"按钮，选择JSON文件导入数据

## 数据格式

导出的JSON文件包含以下字段：
- `lists`: 所有随机列表及其项目
- `settings`: 应用设置（权重递减等）
- `currentListId`: 当前选择的列表ID
- `history`: 最近的抽取历史
- `exportTime`: 导出时间戳
- `version`: 数据版本

## 技术实现

- HTML5 + CSS3 + JavaScript (ES6+)
- 使用 localStorage 进行数据持久化
- 响应式设计，支持各种屏幕尺寸
- CSS 动画和过渡效果

## 浏览器兼容性

支持所有现代浏览器：
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 本地使用

直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何软件或服务器。
